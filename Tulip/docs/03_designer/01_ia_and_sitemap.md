## 1. 문서 목적

Planner의 PLN-08 IA 초안(식별 화면 86개)을 발전시켜 **확정 사이트맵**, **메뉴 ID 체계**, **권한별 메뉴 트리**, **네비게이션 정책**을 정의한다. 본 문서는 와이어프레임·컴포넌트 라이브러리·프론트엔드 라우팅 설계의 기준이 된다.

## 2. 사용자 권한 모델

| 권한코드 | 역할 | 진입 도메인 | 주 디바이스 | 메뉴 노출 정책 |
|---|---|---|---|---|
| **SA** | 시스템관리자 (플랫폼 운영자) | 관리자웹 (전 메뉴 + 테넌트) | 데스크톱 | 풀세트 + 테넌트관리 |
| **DIR** | 관장 | 관장 대시보드, 통계, 결재 | 데스크톱·태블릿 | 대시보드 우선 + 결재/통계 노출 |
| **LIB** | 사서 (편목·수서·열람 전반) | 관리자웹 전 도메인 | 데스크톱 | 풀세트 (테넌트 제외) |
| **CIR** | 일반 사서 (대출·반납 전담) | 열람·회원·시설 | 데스크톱·POS | 열람 중심 메뉴 단순화 |
| **USR** | 일반 이용자 | OPAC | 모바일·데스크톱 | OPAC 메뉴 |
| **EXT** | 외부 방문자 (임시증) | OPAC 검색 한정 | 모바일·키오스크 | 검색만 |

권한 분기 처리는 메뉴 정의 JSON 의 `roles: ["SA","LIB"]` 필드로 제어하며 라우트 가드와 사이드바 렌더링이 동일 데이터를 참조한다.

## 3. 메뉴 ID 체계

```
M-{도메인}-{대분류번호}-{중분류번호}-{상세번호}
```

| 도메인 코드 | 의미 |
|---|---|
| CMN | 공통 (대시보드/회원/코드/권한/정책) |
| ACQ | 수서 |
| CAT | 목록 (편목/KORMARC) |
| CIR | 열람 (대출/반납/예약) |
| COL | 장서 관리 |
| ACS | 출입 관리 |
| FAC | 시설 관리 |
| STA | 통계·리포트 |
| SYS | 시스템 (테넌트·외부연동·감사) |
| OPC | OPAC (이용자 포털) |
| KSK | 키오스크 |

예시: `M-CAT-03-02` = 목록(CAT) 대분류 3번 > 중분류 2번 = KORMARC 편집기.

화면 ID(`SCR-...`)와 메뉴 ID(`M-...`)는 1:N 관계(메뉴 1개에 화면 다수) 매핑 표를 `5.화면-메뉴 매핑 매트릭스`에서 제공한다.

### 4.2 관장(DIR) 대시보드 (관리자웹 내 별도 진입)

관장은 관리자웹과 동일 도메인을 쓰되 진입 시 KPI 중심의 별도 레이아웃으로 라우팅된다.

```
[관장 모드]
├─ M-DIR-01 임원 대시보드
│  ├─ M-DIR-01-01 운영 KPI (대출·방문·예산 집행률)
│  ├─ M-DIR-01-02 도메인별 요약 (수서/장서/열람/시설)
│  └─ M-DIR-01-03 이상 징후·알림
├─ M-DIR-02 통계·리포트 (M-STA-09 공유)
├─ M-DIR-03 결재함 (예산·제적·폐기·기증 승인)
└─ M-DIR-04 공지·내정보
```

## 6. 네비게이션 정책

### 6.1 글로벌 네비게이션 (GNB) — 관리자웹

| 위치 | 요소 | 동작 |
|---|---|---|
| 좌측 상단 | Tulip+ 로고 | 클릭 시 통합 대시보드 (`M-CMN-01-01`) 이동 |
| 좌측 사이드바 | 1·2 레벨 메뉴 트리 (접힘/펼침) | 활성 메뉴 하이라이트, 키보드 `Alt+1~9` 단축 |
| 상단 헤더 | 관 전환 셀렉터 (다관 테넌트) | 활성 관 변경 시 데이터 컨텍스트 갱신 |
| 상단 헤더 | 글로벌 검색 (`Ctrl+K`) | 회원/서지/소장 통합 검색 팔레트 |
| 상단 헤더 | 알림 종, 사용자 메뉴 | 알림센터·내정보·로그아웃 |
| 본문 상단 | 브레드크럼 | 3레벨까지 표시, 마지막 레벨은 현재 페이지 |

### 6.2 세컨드 네비 (탭·서브탭)

- 도메인 내 관련 화면 묶음에 **상단 탭바**를 사용한다. 예) 회원관리 > [조회 | 등록 | 유형 | 일괄등록 | 통계].
- 단일 화면 내 컨텐츠 그룹 전환은 **세그먼트 탭** 또는 **사이드 탭**(긴 리스트의 경우)을 사용한다.

### 6.3 브레드크럼

```
대시보드  >  목록(편목)  >  KORMARC 편집기
[홈 아이콘]   [대분류 링크]    [현재 페이지·텍스트만]
```

- 절단 규칙: 4레벨 이상 깊이는 `...` 으로 축약.
- 모바일에서는 1단계만(`< 뒤로`) 노출.

### 6.4 퀵메뉴 / 단축키 정책 (사서용)

| 단축키 | 기능 |
|---|---|
| `Ctrl+K` | 글로벌 검색 팔레트 |
| `Alt+1~9` | 대분류 메뉴 직접 이동 |
| `F2` | 대출·반납 카운터로 즉시 진입 |
| `F3` | KORMARC 편집기 열기 |
| `F4` | 회원 검색 모달 |
| `Ctrl+S` | 폼·MARC 저장 |
| `Esc` | 모달·드로어 닫기 |
| `Tab` / `Shift+Tab` | 폼 필드 이동 (포커스 트랩 보장) |

- 사서 사용자 화면 하단에 **단축키 도움말 드로어**(`?` 키) 상시 제공.

### 6.5 OPAC 네비게이션

- 데스크톱: 상단 헤더 = 로고·검색바·MyLibrary·로그인·언어전환.
- 모바일: 하단 탭바 5개(홈·검색·MyLibrary·좌석·메뉴), 햄버거 메뉴는 사용하지 않음.
- 비로그인 검색 허용, 예약·신청 클릭 시 로그인 모달.

### 6.6 다국어·다크모드 토글

- 헤더 우측 사용자 메뉴 하위에 **언어 전환(한/영)**, **테마(라이트/다크/시스템)** 항목 상시 노출.
- 사서 화면은 라이트 기본, OPAC 모바일은 시스템 추종 기본.
