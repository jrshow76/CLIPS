# TradePilot 테스트 케이스 카탈로그 (Test Case Catalog)

> 문서 ID: 51_TEST_CASES
> 버전: v1.0
> 작성자: QA
> 최종 수정일: 2026-05-12

본 문서는 TradePilot의 도메인별 테스트 케이스를 카탈로그 형태로 관리한다. 케이스 ID는 `TC-{도메인}-{NNN}` 규칙을 따른다. 우선순위는 `P0/P1/P2`, 자동화 여부는 `자동/수동`으로 표기한다.

## 표 컬럼 정의
- **ID**: 테스트 케이스 고유 ID (도메인 접두사 + 일련번호)
- **제목**: 케이스 한줄 요약
- **사전조건**: 실행 전 만족해야 할 상태
- **단계**: 실행 순서(번호 매김)
- **예상결과**: 기대 응답/UI 상태
- **우선순위**: P0(차단) / P1(핵심) / P2(보조)
- **자동화**: 자동(스크립트 존재) / 수동
- **연관 FR/에러코드**: 명세서 추적
