# TradePilot 실거래(LIVE) 전환 체크리스트

> 문서 ID: 62_GO_LIVE_CHECKLIST
> 버전: v1.0
> 작성자: BackendSenior, QA
> 검토자: PM, DevLead, 사용자
> 최종 수정일: 2026-05-12

본 체크리스트는 TradePilot을 **모의투자(SIM) → 실거래(LIVE)** 로 전환하기 직전에 PM/DevLead/QA/사용자 4자가 1회 점검하는 사인오프 양식이다.

전제: `60_creon_integration_plan.md` 의 **Stage 3 (모의투자 1주일 연속 운영)** 통과.

전환 후 첫 5영업일은 **소액 한도(일일 50만원)** 로 운영하며, 30일 사후 점검 항목까지 모두 통과해야 정상 운영으로 인정한다.

## 10. LIVE 전환 실행 절차 (전환 당일)

본 절차는 위 9개 섹션이 모두 OK 상태일 때만 진행한다.

```
[T-1시간]
1. PM 최종 승인 확인
2. prelive_smoketest.sh 실행 → 6/6 PASS
3. 운영 채널에 LIVE 전환 안내

[T-30분]
4. 사용자 본인이 본인인증 + OTP 재인증
5. 약관 / 면책 동의 재확인

[T-0]
6. 게이트웨이 환경변수 CREON_TRADE_ENV=REAL 변경
7. 게이트웨이 재기동 (NSSM restart)
8. /readyz com_connected=true 확인
9. /account expected_prefix가 REAL 접두사와 일치 확인
10. 본체에서 사용자 trade_mode를 LIVE로 전환 (7단계 게이트 통과)
11. 첫 주문 1주 발주 → 체결 확인 → audit_log 검증

[T+5분]
12. 헬스비트 정상 수신 확인
13. 일일 손실 모니터링 활성

[T+30분]
14. 첫 자동매매 시그널 발생 (있다면)
15. 모니터 대시보드 30분 시점 스냅샷 저장
```
