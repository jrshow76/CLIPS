## 1. 프로젝트 개요

### 1.1 프로젝트 목표

내가 방문했던 장소를 기록하고 지도 기반으로 시각화하는 웹 애플리케이션을 개발한다.
사용자는 장소 정보(위치, 날짜, 사진, 메모)를 쉽게 등록하고, 카테고리/지역별로 나의 발자국을 되돌아볼 수 있다.

### 1.2 프로젝트 배경

- 여행 및 일상 속 방문 장소를 체계적으로 기록하고 싶은 수요 증가
- 기존 메모 앱·SNS는 장소 중심 아카이빙에 부적합
- 지도 시각화와 통계 기능으로 개인화된 장소 히스토리 서비스 제공

### 1.3 프로젝트 범위

**포함(In-Scope)**
- 사용자 인증 (회원가입, 로그인, 로그아웃, 토큰 갱신)
- 장소 기록 등록/수정/삭제 (위치, 날짜, 사진, 메모)
- 지도 기반 시각화 (방문 장소 마커 표시, 클러스터링)
- 장소 목록/검색/필터링 (카테고리, 날짜, 지역)
- 카테고리 관리 (맛집, 카페, 관광지 등 사용자 정의 포함)
- 통계/분석 (월별 방문 수, 지역별 분포, 카테고리 분포)
- 반응형 웹 UI (PC, 모바일 대응)

**제외(Out-of-Scope)**
- 모바일 네이티브 앱 (iOS / Android)
- 소셜 기능 (팔로우, 공유, 피드)
- 외부 SNS 연동 (소셜 로그인 제외)
- 오프라인 모드
