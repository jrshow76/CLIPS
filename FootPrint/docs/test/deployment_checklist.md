## 5. 보안 검토

- [ ] 인증 없이 보호된 API 접근 시 401 응답 확인

- [ ] SQL Injection 입력값 처리 확인 (TC-SEC-001)
- [ ] JWT Access Token 만료(30분) 시 갱신 흐름 정상 동작
