## 2. 화면 계층 구조

```
[Shelfy 웹 앱]
├── 공개 영역 (비로그인 포함)
│   ├── SCR-001: 홈 (탐색 피드)
│   │   ├── 카테고리 필터 탭
│   │   ├── 정렬 셀렉터 (최신/인기/낮은가격/높은가격)
│   │   └── 상품 카드 그리드
│   ├── SCR-002: 검색 결과
│   │   ├── 검색 필터 패널 (카테고리, 판매유형, 가격범위)
│   │   └── 검색 결과 그리드
│   ├── SCR-003: 상품 상세
│   │   ├── 이미지 갤러리
│   │   ├── 상품 정보 패널
│   │   │   ├── 단일 구매 버튼 (saleType에 따라 표시)
│   │   │   └── 구독 플랜 선택 UI (saleType에 따라 표시)
│   │   └── 셀러 정보 카드 (→ SCR-004 링크)
│   └── SCR-004: 셀러 프로필 / 선반
│       ├── 셀러 기본 정보 헤더
│       └── 공개 상품 그리드
│
├── 인증 영역
│   ├── SCR-010: 회원가입
│   ├── SCR-011: 로그인
│   ├── SCR-012: 이메일 인증 완료 안내
│   ├── SCR-013: 비밀번호 재설정 요청
│   └── SCR-014: 비밀번호 재설정
│
├── 결제 영역 (로그인 필수)
│   ├── SCR-040: 결제 페이지
│   │   ├── 상품 요약 정보
│   │   ├── 구독 플랜 선택 (구독형)
│   │   └── 결제 수단 선택
│   ├── SCR-041: 결제 완료
│   └── SCR-042: 결제 실패
│
└── 대시보드 (로그인 필수)
    ├── SCR-020: 내 대시보드 (홈)
    │   ├── 최근 수익 요약 위젯
    │   ├── 활성 구독자 수 위젯
    │   └── 최근 구매/구독 알림
    ├── SCR-021: 내 선반 관리
    │   ├── 상품 목록 테이블 (상태별 탭)
    │   ├── 공개/비공개 토글
    │   └── 등록/수정/삭제 액션
    ├── SCR-022: 상품 등록
    ├── SCR-023: 상품 수정
    ├── SCR-024: 수익 현황
    │   ├── 기간별 수익 차트
    │   └── 상품별 매출 테이블
    ├── SCR-025: 내 프로필 설정
    ├── SCR-030: 구매 내역
    └── SCR-031: 구독 내역
```

## 3. 주요 흐름도

### 3.1 인증 흐름

#### 3.1.1 회원가입 흐름

```
[사용자 진입]
      |
      v
[SCR-010 회원가입 화면]
  - 이메일, 비밀번호, 닉네임 입력
  - 약관 동의 체크
      |
      |--- [유효성 검사 실패] --> 인라인 오류 메시지 표시 (화면 유지)
      |
      v
[회원가입 API 호출]
      |
      |--- [이메일/닉네임 중복] --> 오류 토스트 표시 (화면 유지)
      |
      v
[인증 이메일 발송 완료 안내 모달]
      |
      v
[SCR-011 로그인 화면으로 이동]
      |
      v
(별도 흐름) [이메일 인증 링크 클릭]
      |
      v
[SCR-012 이메일 인증 완료 안내 페이지]
  - 성공: "인증 완료! 모든 기능을 이용할 수 있습니다." + 로그인 버튼
  - 토큰 만료: "링크가 만료되었습니다." + 재발송 버튼
```

#### 3.1.2 로그인 흐름

```
[SCR-011 로그인 화면]
  - 이메일, 비밀번호 입력
      |
      |--- [유효성 검사 실패] --> 인라인 오류 메시지
      |
      v
[로그인 API 호출]
      |
      |--- [인증 실패 (1~4회)] --> "이메일 또는 비밀번호를 확인하세요." 토스트
      |--- [인증 실패 (5회)] --> "계정이 잠금되었습니다." 안내 모달
      |--- [탈퇴 계정]      --> "탈퇴된 계정입니다." 안내
      |
      v
[로그인 성공]
      |
      |--- [로그인 전 방문 페이지가 있었다면] --> 이전 페이지로 이동 (redirect_to)
      |--- [없다면] --> SCR-020 내 대시보드로 이동
```

#### 3.1.3 비밀번호 재설정 흐름

```
[SCR-013 비밀번호 재설정 요청]
  - 이메일 입력
      |
      v
[재설정 메일 발송 API 호출]
      |
  (이메일 존재 여부 노출 금지 - 보안)
  "해당 이메일로 재설정 링크를 발송했습니다." 항상 표시
      |
      v
(별도 흐름) [이메일 내 재설정 링크 클릭]
      |
      v
[SCR-014 새 비밀번호 입력]
  - 새 비밀번호, 확인 입력
      |
      |--- [토큰 만료] --> "링크가 만료되었습니다." + 재요청 유도
      |
      v
[비밀번호 재설정 완료 안내]
      |
      v
[SCR-011 로그인 화면으로 이동]
```

### 3.2 상품 등록 흐름

```
[SCR-021 내 선반 관리]
  [+ 상품 등록] 버튼 클릭
      |
      |--- [이메일 미인증 사용자] --> "이메일 인증 후 이용 가능" 안내 모달
      |
      v
[SCR-022 상품 등록 화면]
  Step 1: 기본 정보
    - 제목, 설명, 카테고리 입력
      |
      v
  Step 2: 판매 유형 선택
    - PURCHASE / SUBSCRIBE / BOTH 선택
    - 가격 / 구독 플랜 입력
      |
      v
  Step 3: 이미지 업로드
    - 드래그앤드롭 또는 파일 선택
    - 썸네일 지정
    - 태그 입력
      |
      v
  Step 4: 공개 여부 선택
    - DRAFT 저장 또는 즉시 PUBLISH
      |
      |--- [임시저장 (DRAFT)] --> SCR-021 내 선반 관리 (비공개 탭)
      |--- [공개 등록 (PUBLISHED)] --> SCR-003 상품 상세 페이지
```

### 3.3 구매 흐름

```
[SCR-003 상품 상세]
  [구매하기] 버튼 클릭
      |
      |--- [비로그인] --> SCR-011 로그인 페이지로 이동 (redirect_to 포함)
      |--- [본인 상품] --> 버튼 비활성화 (툴팁: "본인 상품입니다")
      |
      v
[SCR-040 결제 페이지]
  - 상품 요약, 결제 금액 확인
  - 결제 수단 선택 (카드 / 카카오페이 / 네이버페이)
  [결제하기] 클릭
      |
      v
[결제 PG 팝업 또는 리다이렉트]
      |
      |--- [결제 성공] --> SCR-041 결제 완료 페이지
      |                    - "구매 완료!" 확인 메시지
      |                    - [내 구매 내역 보기] / [셀러 선반으로 이동] 버튼
      |
      |--- [결제 실패] --> SCR-042 결제 실패 페이지
                           - 실패 사유 안내
                           - [다시 시도] / [홈으로] 버튼
```

### 3.4 구독 흐름

#### 3.4.1 구독 신청

```
[SCR-003 상품 상세]
  구독 플랜 선택 (Basic / Premium 등)
  [구독 시작하기] 버튼 클릭
      |
      |--- [비로그인] --> SCR-011 로그인 페이지로 이동
      |--- [이미 구독 중] --> "이미 구독 중인 상품입니다." 모달
      |
      v
[SCR-040 결제 페이지 - 구독 모드]
  - 선택한 플랜 요약
  - 다음 결제일 안내
  - 정기 결제 수단 선택 (카드)
  [구독 시작] 클릭
      |
      v
[첫 결제 PG 처리]
      |
      |--- [성공] --> SCR-041 구독 완료 페이지
      |               - "구독이 시작되었습니다!" 안내
      |               - 다음 결제일, 해지 방법 안내
      |
      |--- [실패] --> SCR-042 결제 실패 페이지
```

#### 3.4.2 구독 해지

```
[SCR-031 구독 내역]
  활성 구독 카드 선택
  [구독 해지] 버튼 클릭
      |
      v
[해지 확인 모달]
  - "현재 기간 만료 후 해지됩니다."
  - 만료일 표시
  [해지 확인] / [취소]
      |
      v
[구독 해지 API 호출]
      |
      v
[구독 상태: CANCEL_REQUESTED 로 변경]
  - 화면 갱신: "해지 예정" 상태 표시
  - 해지 취소 버튼 노출
```

### 3.5 글로벌 네비게이션 흐름

```
[GNB - 글로벌 네비게이션 바]
  ├── [Shelfy 로고] --> SCR-001 홈
  ├── [검색창] --> 입력 시 SCR-002 검색 결과 (실시간 쿼리)
  ├── [카테고리] --> SCR-001 (카테고리 필터 적용)
  │
  ├── [비로그인 상태]
  │     ├── [로그인] --> SCR-011
  │     └── [회원가입] --> SCR-010
  │
  └── [로그인 상태]
        └── [내 계정 드롭다운]
              ├── [내 대시보드] --> SCR-020
              ├── [내 선반] --> SCR-021
              ├── [구매 내역] --> SCR-030
              ├── [구독 내역] --> SCR-031
              ├── [프로필 설정] --> SCR-025
              └── [로그아웃] --> 로그아웃 처리 후 SCR-001
```

## 4. 화면별 접근 권한 매트릭스

| 화면 ID | 화면명 | 비로그인 | 로그인(미인증) | 로그인(인증완료) | 셀러 본인 |
|---|---|:---:|:---:|:---:|:---:|
| SCR-001 | 홈 | O | O | O | O |
| SCR-002 | 검색 결과 | O | O | O | O |
| SCR-003 | 상품 상세 | O | O | O | O |
| SCR-004 | 셀러 프로필 | O | O | O | O |
| SCR-010 | 회원가입 | O | - | - | - |
| SCR-011 | 로그인 | O | - | - | - |
| SCR-012 | 이메일 인증 | O | O | - | - |
| SCR-013 | 비밀번호 재설정 요청 | O | O | O | O |
| SCR-014 | 비밀번호 재설정 | O | O | O | O |
| SCR-020 | 내 대시보드 | - | O | O | O |
| SCR-021 | 내 선반 관리 | - | O | O | O |
| SCR-022 | 상품 등록 | - | X (안내) | O | O |
| SCR-023 | 상품 수정 | - | - | - | O |
| SCR-024 | 수익 현황 | - | O | O | O |
| SCR-025 | 내 프로필 설정 | - | O | O | O |
| SCR-030 | 구매 내역 | - | O | O | O |
| SCR-031 | 구독 내역 | - | O | O | O |
| SCR-040 | 결제 페이지 | 로그인 유도 | O | O | - (본인 불가) |
| SCR-041 | 결제 완료 | - | O | O | O |
| SCR-042 | 결제 실패 | - | O | O | O |

범례:
- O: 접근 가능
- -: 해당 없음 또는 자동 리다이렉트
- X (안내): 접근은 되나 기능 제한 안내 표시
